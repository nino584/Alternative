# Alternative Concept Store

A React + Vite frontend for the Alternative concept store — curated luxury pre-order fashion (Tbilisi, Georgia / Shanghai sourcing).

## Stack

- React 18
- Vite 5
- React Router 6
- Local storage persistence for cart, wishlist, and language

## Getting Started

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
